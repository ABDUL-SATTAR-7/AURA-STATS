## 📌 Overview

**AuraStats** is an AI-powered gaming analytics platform that transforms raw gameplay statistics into meaningful insights. It helps gamers understand their performance, identify strengths and weaknesses, monitor progress over time, and receive personalized recommendations for improvement.

Whether you're a casual gamer or an aspiring esports player, AuraStats provides detailed analytics to help you level up your gameplay.

### 📊 Performance Dashboard

- Accuracy tracking

- Statistical summaries
